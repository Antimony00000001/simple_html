现代CSS全方位入门教程

CSS (Cascading Style Sheets) 是用于描述HTML文档外观和格式的语言。本教程将通过交互式示例，带你掌握从基础到高级的现代CSS技术。

1. 介绍与语法

CSS用于为HTML元素“穿上”漂亮的衣服。其基本语法由一个选择器和一个声明块组成。声明块包含一个或多个由分号分隔的属性:值对。

/* 这是一个CSS注释 */
p {
  color: blue; /* 将所有p标签的文字颜色设为蓝色 */
  font-size: 16px;
}

添加CSS的三种方式

  1. 内部样式表: 使用 <style> 标签放在 <head> 内。(本教程使用的方式)
  2. 外部样式表: 将CSS代码保存为 .css 文件，使用 <link> 标签引入。 (推荐方式)
  3. 内联样式: 直接在HTML元素的 style 属性中书写。(不推荐，难以维护)

3. 盒模型 (Box Model)

在CSS中，每个元素都被看作一个矩形的盒子。盒模型描述了这些盒子是如何构成的。

它由内到外包括：内容(content), 内边距(padding), 边框(border), 和外边距(margin)。

Content

box-sizing: border-box; 是现代CSS布局的最佳实践，它让元素的宽高计算更直观。设置width和height时，padding和border将被包含在内。
